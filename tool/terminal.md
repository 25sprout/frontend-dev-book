# Terminal 指令

### 設定 localhost
用 `php` 開
```bash
php -S localhost:8000
php -S 0.0.0.0:8000 # 也可以開內網
```
用 `node` 的 package `http-server` 開
```bash
npm i -g http-server
http-server -p 8000
```
其他：
- python simplehttpserver
- [webpack-dev-server](https://webpack.github.io/docs/webpack-dev-server.html)
